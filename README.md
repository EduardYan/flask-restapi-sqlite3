# Flask Sqlite RestAPI.

__This is a restapi made with python and flask, using as sqlite as database.__


# Dependencies.

__Install dependecies with requirements file:__

```bash
$ pip3 install -r requirements.txt
```

# Run.

```bash
$ python3 index.py
```

# Routes.

* GET / or /books
* GET /books/1
* POST /books
* PUT /books/1
* DELETE /books/2
